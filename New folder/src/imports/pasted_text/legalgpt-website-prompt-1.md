**Figma Design Prompt / Brief for LegalGPT Website UI/UX**

Design a complete minimalist, formal, trustworthy website UI/UX for **LegalGPT** — an AI-powered legal assistant platform built for Indian users (Indian law advisor firm theme). The website must feel professional, secure, credible, and calm — like a premium Indian law firm digital presence combined with modern AI elegance.

**Overall Style & Aesthetic Guidelines**
- Theme: Light theme only — predominantly white/off-white background (#FFFFFF or #FAFAFA / #FCFCFC) with subtle soft grays (#F5F5F5, #EDEDED, #D4D4D4) for sections/dividers.
- Primary accent color: Deep navy blue (#1E3A8A or #111827) or professional Indian-legal teal-blue (#0F766E / #065F46) — use sparingly for buttons, links, icons, and highlights.
- Secondary accents: Soft gold or muted saffron (#D97706 / #B45309 very desaturated) for subtle trust elements (e.g., checkmarks, borders on key CTAs) to nod to Indian professionalism without being loud.
- Typography: Clean, modern sans-serif fonts — use Inter or similar system font stack (fallback: system-ui), with headings in slightly bolder weight. Font sizes generous for readability (body 16–18px, headings 36–60px on desktop). Excellent line height (1.5–1.7) and letter spacing for formality.
- Border radius: Generously rounded corners (16–24px) on cards, buttons, input fields, modals — soft and approachable, never sharp or boxy.
- Shadows: Very subtle soft shadows (minimal elevation, rgba(0,0,0,0.04–0.08)) — almost flat but with gentle depth for hierarchy.
- Spacing & Layout: Extremely generous white space / padding. No tight grids, no boxy layouts, no heavy card grids. Use fluid, organic flow with large breathing room between sections. Avoid strict 12-column grids — prefer asymmetric, centered, or flowing single-column-ish layouts on desktop with natural alignment.
- No heavy shadows, no skeuomorphism, no gradients unless very subtle. Keep it airy, minimal, elegant, trustworthy — think high-end Indian corporate law firm site crossed with modern AI SaaS (like clean versions of Clio light theme or minimalist legal sites from 2025 trends).
- Icons: Minimal line icons (Stroke 1.5–2px), blue/navy tinted, sparse usage.
- Imagery: High-quality subtle stock photos — Indian professionals in formal attire (diverse, inclusive), justice scales (stylized/minimal), open law books, secure document icons, AI + gavel symbolic art (very abstract/minimal). No cheesy AI robots. Use images sparingly — mostly section backgrounds or small hero visuals.
- Accessibility: High contrast (WCAG AA+), focus states visible (blue outline), semantic structure.

**Key Pages / Screens to Design (Mobile-first responsive, show desktop + mobile views)**
1. **Landing Page (Home) – Minimal & focused**
   - Hero section: Large centered headline “Your AI Legal Advisor – Powered by Indian Law Expertise”, subheadline explaining RAG + LLM for document analysis, drafting, Q&A. One prominent CTA button: “Start Free Trial” or “Upload Document & Chat”.
   - Subtle background image or abstract legal/AI visual (low opacity).
   - Below hero: 3–4 soft rounded feature cards (no grid feel — stacked or gently overlapping with flow): Document Upload & Analysis, AI Chat for Legal Q&A, Instant Drafting (Contracts/NDA/Letters), Advocate Search & Secure Email.
   - Trust bar: “Secure • Accurate • 24/7 • Built for Indian Laws” with tiny icons.
   - Footer minimal: Links, copyright, privacy policy.

2. **Chat / AI Interface Page (Main tool after login)**
   - Clean sidebar (collapsible on mobile) for uploaded documents list.
   - Main area: Chat window with rounded message bubbles (user right-aligned light gray, AI left-aligned white with subtle blue border).
   - Input bar at bottom: Rounded, large, with upload icon + send.
   - Top bar: Document title / context shown, clear buttons for “Extract Clauses”, “Draft Letter”, “Generate PDF”.

3. **Document Upload & Analysis Screen**
   - Drag-drop zone: Large rounded dashed border area.
   - Supported formats badge: PDF, DOCX.
   - Preview thumbnail of uploaded doc (rounded).
   - “Analyze with AI” big rounded button.

4. **Document Generation / Drafting Screen**
   - Template selector: Soft rounded cards for Contract, NDA, Legal Notice, etc.
   - Form fields to fill context (rounded inputs).
   - Live preview pane with generated text.
   - “Download PDF” / “Send via Email” buttons.

5. **Advocate Search Module**
   - Search bar (large rounded).
   - Filters: Specialty (dropdown), Location (Indian cities), Experience.
   - Results: Vertical list of advocate cards — photo (rounded), name, specialty, location, “Contact” button.

6. **User Dashboard / Profile (after login)**
   - Minimal sidebar navigation: Dashboard, My Documents, Chat History, Advocate Search, Settings.
   - Greeting + quick stats (documents analyzed, drafts created).
   - Recent activity feed (rounded cards).

7. **Authentication Screens**
   - Login / Sign Up: Centered form, rounded inputs, “Login with Google” option, subtle Indian legal trust note.

**Additional Requirements**
- Navigation: Simple top bar — Logo left, minimal links (Home, Features, How it Works, Pricing, Contact), right side CTA “Login” / “Get Started”.
- Buttons: Large, rounded (24px+ radius), solid fill on primary (navy/teal), outline on secondary. Hover: subtle lift + color shift.
- No heavy animations — only micro-interactions (fade-ins, smooth hover).
- Mobile: Fully responsive — stack everything vertically, hamburger menu, touch-friendly large tap targets.
- Indian touch: Use phrasing like “Advocate Search”, “Legal Notice”, “ compliant with Indian laws”, subtle tricolor-inspired accents only if tasteful (avoid flag feel).

Create high-fidelity frames for: Home (desktop + mobile), Chat interface (desktop), Upload screen, Dashboard, Advocate search results. Use auto-layout, components for buttons/cards/inputs to keep consistent.

Focus on calm, trustworthy, professional minimalism — users should feel they are dealing with a credible Indian law advisor firm, not a flashy tech startup. Prioritize clarity, security signals, and ease over visual excitement.
